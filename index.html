About Me

I'm an alumna of the Microsoft Research DS3 program, where I created statistical models and designed algorithms to fix the uneven distribution of bike-sharing programs worldwide. Projects

Microsoft Research: Self-Balancing Bikes

A data analysis project to create a rerouting scheme for riders to self-balance the Citi Bike system. We anticipate that we can decrease vehicle transportations by offering financial incentives to take bikes from relatively full stations and return bikes to relatively empty stations (with rerouting advice provided via an app). We used publicly available data obtained via the CitiBike website, consisting of starting and ending locations, times, and user characteristics for each trip taken from July 2013 through May 2014. Using this dataset, we estimated CitiBike traffic flow, which enabled us to build agent-based simulation models in response to incentives and rerouting information. By estimating various parameters under which to organize incentive schemes, we found that such a program would help to improve CitiBike's environmentalism and increase productivity, as well as being financially beneficial for both CitiBike and its riders. I presented this research at the 2014 Association of Computing Machinery's SIGKDD at Bloomberg conference and 2015 ACM Richard Tapia Celebration of Diversity in Computing conference, where we were awarded 2nd place nationwide in Undergraduate Research. See our extended abstract, video presentation, or view it on Github.

Federal Reserve Board of Governors: Time Series ETL Dashboard

The time series dashboard was developed in order to visualize and display the efficiency of internal time series data release ETL processes within the Office of the Chief Data Officer at the Federal Reserve Board. The dashboard KPIs were split into the categories of 'timeliness' and 'quality' and required implementation of new data collection features within each set of series per data release. I ran preliminary statistics on the status of current releases and presented my findings and recommendations for data rearchitecture to division-wide management. View it on Github.

Aalto University Design Factory: ABB Helm

Helm is a ship control interface which seeks to revolutionize modern Azipod-powered vessels. By digitizing the manuals and streamlining the information display, we created an ergonomic and intuitive user experience. As a technical engineer, I built servers, wrote Arduino code, and interfaced a JavaScript web application. Helm was awarded 3rd Place through the Aalto University Product Development Project and sponsored by ABB.

Data For Development: Senegal

A data for development challenge sponsored by Orange Telecommunications and MIT Media Lab. Worked with a team of researchers and professors from US and Senegal to use anonymized mobile datasets provided by Orange to improve the social welfare of the Senegalese people. Our focus was on health and migration in Senegal during significant religious festivals. This project involved conducting extensive background research, data analysis using R, application of machine learning models, and CartoDB for data visualization. Results of the challenge were presented at the NetMob 2015 conference at the MIT Media Lab in Boston, MA. View it on Github.

Visualization of URL Scalability

A script written to parse and analyze a streaming JSON dataset. The project focuses on the scalability of URL sharing by visualizing the growth rate of all non-repeat clients clicking the most shared user hash ID within a 10-minute interval. In order to do this, I selected the most active user hash ID and filtered all non-repeat clients for visualization formatting. I then imported geocoder data through Google Maps API calls in order to generate the final plot. View it on Github.

Square Code Camp: The Generations

The Generations platform is a Ruby on Rails web application created to connect and sustain a network of code camp alumni. We used geo-location data to map the current location of code camp alumni and school, generation year, and interest data as filters for each participant. View it on Github.
