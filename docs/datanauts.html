NASA Datanauts

The NASA Datanauts initiative is an international community of developers, data scientists, designers, students, entrepreneurs, and community organizers looking to further develop their data science skills. Admission to the Datanauts class provides access to a collaborative community and the opportunity to contribute to technical projects, take workshops, and build mentorships within NASA's data science team. Some of the topics covered include:

  • Machine learning/NLP
  • Web-based space mission visualizations
  • Orbital dynamics
  • Text analytics
  • 3D graphics
  • Knowledge architecture
  • Binary applications
  • VR applications
  • Web scraping/crawling
  • Deep learning
  • Data visualizations
