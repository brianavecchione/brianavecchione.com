NASA Datanauts

The NASA Datanauts initiative is an international community of developers, data scientists, designers, students, entrepreneurs, and community organizers looking to further develop their data science skills. Admission to the Datanauts class provides access to a collaborative community and the opportunity to contribute to technical projects, take workshops, and build mentorships within NASA's data science team. Some of the topics covered include:

  • Machine learning/NLP
  • Web-based space mission visualizations
  • Orbital dynamics
  • Text analytics
  • 3D graphics
  • Knowledge architecture
  • Binary applications
  • VR applications
  • Web scraping/crawling
  • Deep learning
  • Data visualizations
  • Hardware
Below are a few featured projects from the program:

Elliptical Orbit Simulator

This project was completed using a NASA Datanaut toolkit. The toolkit teaches readers how to produce Web-based Space Mission Visualizations (WSMV) using the R programming language. First, I read through documentation describing the procedure of specifying a 2D ellipse and applying 3D rotations to it using Keplarian parameters. Then, I specified a 2D ellipse and applied pitch, yaw, and roll rotation matrices. This is done to orient the ellipse in 3D.

Supplemental Documentation
  • 2D geometry
  • 3D orbits

The project was completed in four parts: First, I generated a flat surface ellipse. Next, I transformed the ellipse into an orbit using R package rgl to program three rotations (pitch, yaw, roll). Next, I plotted an interactive image of the ellipses using the rglwidget function. Finally, I plotted an orbit around a sphere. The code is on GitHub.

Starry Night

NASA Datanauts were involved in beta testing, iteration, and expansion of Dr. Kate Stone's "Starry Night" storyboards. The storyboards are DIY kits of piano-like instruments that emit a series of space-themed audio recordings. The construction of Starry Night requires the following materials:

  • Starry Night printed conductive circuit (piano or spot design)
  • Your substrate (we use a foam board)
  • Printed Artwork (or you can design your own)
  • Printed out cutter template
  • A craft knife, ruler and cutting mat - Spray adhesive
  • A 2016 coin cell battery

First, a user must assemble the project using the documents provided below. Then, the user must configure the Novalia Builder application with their conductive circuit to enable their galactic instrument. The project has been since been brought to several international conferences on behalf of the NASA Datanauts community.

Near-Earth Comets API

I created a public Firebase API Using NASA's Near-Earth Comets - Orbital-Elements dataset. To do this, I exported the dataset, set up a Firebase project, imported the JSON file to a Comet Database, configured authorization, and published the project. The API can be accessed here.
